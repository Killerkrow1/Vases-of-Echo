<center><h3>A mod that adds vases containing loot to Ancient Cities.</h3></center>

# 

# Features

- Ancient Cities now contain vases filled with loot. Break the vases to collect the loot, but be careful! Breaking them is very loud and it will alert Sculk Sensors.
- Vases can also be broken with Silk Touch if you want to use them as decoration.
- Vases come in three different sizes: Big, Small, and Tall; all containing the same amount of loot.

# Known Issues
- This mod will break when used with anything else that modifies Ancient City generation.
